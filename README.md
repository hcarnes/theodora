## Installation

Clone this repository

## Google Login
To support logging in with Google, create a `.env` file in the project root with your OAuth2 credentials:
```
GOOGLE_CLIENT_ID=CLIENT_ID_GOES_HERE
GOOGLE_CLIENT_SECRET=SECRET_GOES_HERE
```
